# One Message Per Chart

## Definition

A chart's effectiveness decreases with complexity. The principle demands that each visualization convey exactly one takeaway—one argument, one insight, one implication—so the reader grasps it immediately without mental effort. If your data supports multiple conclusions, create multiple charts, each highlighting the relevant data for its specific point.

## In the Book

McKinsey emphasizes simplicity in presentation. The more complex a chart becomes, the less effective it is. The book states plainly: "The meaning of a chart should be immediately obvious to the reader." If you need to use the same data to make two different points, redraw the data and highlight different elements in each version.

McKinsey's approach is deliberately conservative: minimal color, no 3-D unless necessary, no visual flourishes that distract from the message. Sylvia Mathews, preparing presentations for the President, kept the one-message principle foremost. A chart that requires explanation is a failed chart; it should read at a glance. This applies whether you're using black-and-white slides or "rainbow-hued, three-dimensional computer animations with musical accompaniment"—the visual should never obscure the message.

## Why It Matters

One message per chart enforces discipline on the presenter: it forces you to articulate precisely what each exhibit proves, preventing intellectual sloppiness. For the audience, it respects their cognitive load. They can absorb one insight per image, then move to the next. This principle scales across domains—dashboards, academic posters, infographics, annual reports—anywhere visual communication matters. It also makes redesign easy: if a chart is confusing, you haven't yet figured out which single point it should carry.
