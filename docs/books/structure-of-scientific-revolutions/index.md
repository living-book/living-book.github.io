# The Structure of Scientific Revolutions — Companion

Thomas Kuhn's 1962 essay — the most cited academic book of the twentieth century — replaced the textbook image of science as steady accumulation with a cycle: **normal science → anomaly → crisis → revolution → new normal science**. It gave the world "paradigm shift" and forced every field since to ask which stage of the cycle it's in.

Each chapter page includes an audio overview, a start infographic (read before), and a transformation infographic (read after).

## The Kuhn Cycle

**Normal science** (chapters 2–5) → **anomaly** (6) → **crisis** (7–8) → **revolution** (9–10) → **erasure and progress** (11–13).

## Master Themes

- **Exemplars over rules** — communities cohere on shared examples, not stated criteria
- **Precision manufactures surprise** — only sharp expectations make anomalies visible
- **No rejection without replacement** — choice is always between paradigms
- **Trained perception** — expertise restructures what practitioners literally see
- **Progress without a destination** — puzzle-solving power grows; truth-approach is optional
