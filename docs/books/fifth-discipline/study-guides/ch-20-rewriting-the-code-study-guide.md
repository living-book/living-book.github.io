# Study Guide: Chapter 20 — Rewriting the Code

## Core Idea
The subconscious, not the conscious mind, is what handles detail complexity (driving, tennis, Mozart) — and it can be retrained by practice to structure reality in feedback circles instead of linear chains, the same way one becomes fluent in a new language.

## Key Terms
Detail complexity vs. subconscious capacity · subconscious training · linear vs. systemic language

## Case Summary
Driving 60mph in traffic while conversing, tennis, and playing a Mozart sonata all show the subconscious handling detail complexity the conscious mind can't. Language's bias toward "A caused B" (not "B also caused A") subconsciously teaches linear causality; systemic language retrains it.

## Application Checklist
- [ ] Treat systems-thinking skill as something built by repeated practice (microworlds, real problems), not just reading
- [ ] Notice when your own language ("X caused Y") is hiding a feedback loop running the other way too
- [ ] Watch for humanity's evolutionary bias toward sudden-threat detection blinding you to slow, structurally complex risks
